发表于 2019-03-26 | | 本文总阅读量 次

doCheck

这个函数用于判断是否已经滚动到底部，可以说是整个插件的核心逻辑。由于设置oveflow的元素可以是宿主元素自身，也可以是宿主元素的某个父元素，所以判断会分成两个分支。

这里涉及到了多种尺寸值，包括scrollTop、offsetTop、clientHeight、scrollHeight等等，如果不清楚的话整个函数的逻辑就很难看懂，关于它们的具体意义可以参考我之前写的一篇博客。
